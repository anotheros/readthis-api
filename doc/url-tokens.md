## 根据用户域名获取用户

### HTTP 请求

```
GET  /v2/url-tokens/:token
HEAD /v2/url-tokens/:token
```

### URL 参数

参数名     | 值类型      | 描述
--------- | ---------- | -------------------------------------------------------
token     | string{32} | 个性域名

### 请求体

无

### 响应体

302 跳转至 `/users/:id`
